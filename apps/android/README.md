# Android Project Entry

`apps/android` is the Android project root for external validation and IDE import.

- Gradle root: `apps/android`
- App module: `apps/android/app`
- App entry source: `apps/android/app/src/main/java`
- Shared UI source: `src/ui/mobile/android`
- Platform source: `src/platform/android`

Build from the repository root with:

```bash
./scripts/build_android.sh --debug
```
