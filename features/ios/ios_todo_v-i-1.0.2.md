# iOS Todo v-i-1.0.2

## 当前阶段

在 `v-i-1.0.1` 的基础上，iOS 代码已经从静态 UI 壳层升级为接近可运行的应用骨架。

当前已落地：

- `AVPlayer` 播放容器
- 系统 `fileImporter` 导入入口
- 应用沙盒路径服务与导入资源仓库
- 导入资源持久化到本地 `resources.json`
- 最近播放与播放进度保存
- `xcodegen` 工程描述文件
- 面向 `core` 状态对齐的 iOS 侧桥接快照结构

## 下一步待办

- 在完整 Xcode 环境中生成并运行 `AIPlayerIOS.xcodeproj`
- 将播放器状态改为真实驱动字幕与翻译 UI，而不是占位文本
- 接入字幕解析和字幕绑定流程
- 接入在线翻译 service 与错误降级状态
- 补充资源详情页、删除资源、重新绑定字幕等操作
- 评估是否将 iOS 共享模型进一步下沉出 `ui/mobile/ios`

## 风险说明

- 当前环境缺少完整 Xcode，尚未执行真机或模拟器验证
- `AVPlayer` 已接入，但字幕与翻译仍为占位逻辑
- 当前 `xcodegen` 仅提供工程描述文件，生成工程仍依赖本地安装 `xcodegen`
