# Android Todo v-a-1.0.3

## 版本目标

`v-a-1.0.3` 作为 `v-a-1.0.2` 之后的下一个小版本，聚焦把当前 Android 应用骨架从“本地状态可用”继续推进到“真实媒体能力开始接入”的阶段。

## 当前基础

当前仓库已经具备：

- Android Gradle 工程与 `androidApp` 模块
- Browser / Launch / Player / Settings 四条主路径
- 系统文件选择器接入
- 最近播放本地持久化
- 设置本地持久化
- 播放器页的字幕、翻译、样式、更多面板骨架
- Android 播放控制器接口与预览控制器占位实现

## 下一步开发项

1. 接入真实播放器容器，替换 `PreviewAndroidPlayerController`
2. 将视频 URI 真正绑定到播放器加载流程
3. 接入外挂字幕解析与当前播放会话绑定
4. 为翻译面板接入真实在线翻译请求链路
5. 增加最近播放的播放进度更新与恢复逻辑
6. 增加文件失效后的重新授权恢复链路
7. 增加 Gradle Wrapper，降低 Android 工程上手门槛
8. 增加基础 UI 测试与关键页面截图校验

## 风险与依赖

- 真实播放器路线仍需最终确定
- `core` 与 Android UI 的数据桥接仍未完成
- 真机权限与 URI 生命周期问题需要专项验证
